# Site Builder（Electron版）

GUIでWebサイトをデザインし、静的HTML または Laravel Blade を出力するデスクトップアプリです。
PHPやNode.jsのサーバーは不要。アプリ単体で動作します。

## 開発時の起動

```bash
cd electron-app
npm install      # 初回のみ（electron をダウンロード）
npm start        # アプリ起動
npm run dev      # 開発者ツール付きで起動
```

## 配布用ビルド

```bash
npm run build        # 現在のOS向け
npm run build:win    # Windows用 .exe
npm run build:mac    # Mac用 .dmg
```

`dist/` フォルダに成果物が出ます。

## クイックスタート（最初の1ページを作る）

1. **要素を置く** — 左の「ツール」パネルの `＋ ボタン` `＋ テキスト` `＋ 図形` などを押すと、キャンバス中央に要素が追加されます。
2. **画像を入れる** — PC のフォルダから画像ファイルをキャンバスへ直接**ドラッグ＆ドロップ**。
3. **配置・サイズを整える** — キャンバス上で要素をドラッグして移動、四隅のハンドルでリサイズ。近くの要素やガイドに自動で吸着します（`Alt` を押している間は吸着オフ）。
4. **見た目を編集** — 要素を選ぶと右の「プロパティ」パネルが開き、色・文字・角丸・影・グラデーションなどを変更できます。テキストやボタンは**ダブルクリック**で文字を直接編集。
5. **ページ設定** — 「プロジェクト設定」でプロジェクト名・キャンバスサイズ・背景色・SEO（タイトル/説明/OG画像）を設定。
6. **プレビュー** — 上部の `▶️ Play` で実際の表示を確認（`■ 編集に戻る` で戻る）。
7. **出力** — 「ツール」パネル下部の `🌐 静的サイト出力` または `⚙️ Laravel出力` で書き出し。保存先フォルダを選ぶと一式が生成されます（`Ctrl+S` でも保存して書き出し）。

## 画面構成

パネルはドッキング式で、上部メニュー「表示」から各パネルの表示/非表示を切り替えられます（「レイアウトを初期状態に戻す」で既定配置へ）。

| パネル | 役割 |
|---|---|
| ツール | 要素・コンポーネントの追加、サイト出力 |
| ページ | ページの追加、フォルダ分類 |
| エクスプローラー | 現在ページの要素（レイヤー）一覧・並び順・表示切替 |
| キャンバス | デザインの編集領域（上部に Play／PC・Mobile／定規／ズーム） |
| プロジェクト設定 | プロジェクト名・出力タイプ・背景色・SEO・キャンバスサイズ |
| プロパティ | 選択中の要素の位置・見た目・フォーム設定・整列 |

---

## ツール（追加できる要素）

**基本要素**

- ＋ ボタン … リンク or フォーム送信ボタン
- ＋ 入力欄 … テキスト／メール／電話／数値／テキストエリア
- ＋ テキスト … 見出し・本文
- ＋ 図形 … 四角 / 丸 / 三角
- 画像 … ファイルをキャンバスへドラッグ＆ドロップ

**コンポーネント（複数要素のまとまり）**

- Hero（見出し）／ Card（カード）／ FAQ（Q&A）
- Slider（画像スライダー）／ ArticleGrid（記事一覧）／ Accordion（開閉Q&A）

---

## プロパティ（右パネルでできる編集）

要素を選択すると編集できます。

- **配置** … レイヤー名、X / Y 座標、幅 W / 高さ H、不透明度
- **見た目** … 背景色・文字色（透明度対応）、角の丸み、枠線（太さ・色）、テキスト内容・フォント・揃え
- **グラデーション** … 種類・方向・2色指定
- **影／レイヤースタイル** … ドロップシャドウ（X/Y・ぼかし・広がり・色）、内側シャドウ、光彩、ベベル
- **アニメーション** … フェードイン等の表示アニメ
- **フォーム設定** … ボタンを「送信ボタン」にして送信先・メソッド・完了メッセージを設定、入力欄に name・種類・必須を設定
- **整列・分布** … 複数選択して 左/中央/右・上/中/下 揃え、横/縦の等間隔配置
- **グループ化** … 複数要素をまとめて「フォルダ化（グループ化）」／解除。ワープ（自由変形）モードも切替可能

## マウス操作

| 操作 | 動作 |
|---|---|
| クリック | 要素を選択 |
| ドラッグ（空白から） | 範囲選択（囲んだ要素をまとめて選択） |
| `Ctrl` + クリック | 選択に追加／解除（複数選択） |
| `Shift` + クリック | 直前に選んだ要素との間をまとめて範囲選択 |
| ドラッグ（要素） | 移動（他要素・ガイドへ自動吸着） |
| 四隅・辺のハンドル | リサイズ・回転 |
| ダブルクリック | テキスト／ボタンの文字をその場で編集 |
| `Space` + ドラッグ / ホイールクリックでドラッグ | キャンバスをパン（スクロール） |
| `Ctrl` + ホイール | ズーム |

## キーボードショートカット

| キー | 動作 |
|---|---|
| `Ctrl` + `S` | 保存して書き出し |
| `Delete` / `Backspace` | 選択要素を削除 |
| `Ctrl` + `C` / `Ctrl` + `V` | コピー／貼り付け |
| `Ctrl` + `Z` | 元に戻す |
| `Ctrl` + `Shift` + `Z` / `Ctrl` + `Y` | やり直し |
| `Alt`（ドラッグ中に押す） | 吸着（スナップ）を一時的にオフ |

> Undo/Redo はページごとに独立した履歴を持ちます。

## ページとフォルダ

- 「ページ」パネルの `＋ ページを追加` で複数ページを作成できます。ページはクリックで切り替え。
- `📁 フォルダを追加` でページを分類できます。出力時、フォルダ名はそのまま出力先のサブフォルダになります。
- SEO は「サイト共通」と「ページ個別」の2段構え。ページ個別が空なら共通値が使われます。

---

## レスポンシブ（PC／スマホ）

- キャンバス上部の `💻 PC` / `📱 Mobile` で表示を切り替えます。
- PC で配置した内容は自動でスマホにも反映されます。スマホ表示で要素を動かすと、**その要素だけ**スマホ専用の配置として保存されます（PC の配置は保持）。
- キャンバスサイズは「プロジェクト設定」で PC 用・スマホ用をそれぞれ指定できます。出力 HTML は `@media (max-width: 768px)` でスマホ配置に切り替わります。

---

## 定規とガイド

- キャンバス上部の `📏 定規` で定規・ガイドの表示/非表示を切り替え。
- 上の定規から下へドラッグで**縦ガイド**、左の定規から右へドラッグで**横ガイド**を作成。要素の移動・リサイズ時にガイドへ吸着します。
- ガイドをドラッグで移動、**ダブルクリックで削除**、左上の角をクリックで**全ガイド消去**。

---

## プレビュー

- 上部の `▶️ Play` で、実際の出力に近い表示（アニメーションやスライダーの動作込み）を確認できます。`■ 編集に戻る` で編集に戻ります。

---

## 保存と読み込み

- `Ctrl+S`（またはツールパネルの出力ボタン）で、選んだフォルダに **サイト一式＋ `project.json`** が書き出されます。
- 上部メニュー「ファイル → 開く…」で `project.json` を選ぶと、続きから編集できます。
- 一度保存したプロジェクトは、以降そのフォルダへ上書き保存されます。

---

## 出力タイプ

- 「🌐 静的サイト出力」… `index.html` ＋ `images/`（＋ CSS を別ファイル出力：`css/`）を出力。
  さくらレンタルサーバー等にFTPで置けばそのまま公開可能。
  ボタンのリンク先にGoogleフォームのURLを設定しておけば、
  フォーム送信→スプレッドシート→GASで自動メール、という構成が組める。

- 「⚙️ Laravel出力」… `resources/views/index.blade.php` ＋ `routes/web.php`
  ＋ `public/images/`（＋ `public/css/`）を出力。既存のLaravelプロジェクトに組み込める。

> **CSS の出力先**: 既定で CSS は HTML に埋め込まず `css/` フォルダへ分離出力します（「プロジェクト設定」のチェックで切り替え可能）。ページ共通のアニメーションは `css/common.css`、ページ固有のスタイルは `css/<ページ名>.css` に書き出されます。

### お問い合わせフォームの設計（静的サイトの場合）

```
お問い合わせボタン（リンク先にGoogleフォームURLを設定）
   ↓
Googleフォーム
   ↓
スプレッドシート（回答が自動で蓄積）
   ↓
GAS（フォーム送信トリガー）
   ↓
担当者へ自動メール（件名に氏名を差し込み）
```

    ├── main/

        └── utils/           色などの共通ヘルパー
